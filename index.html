What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree-like hierarchy of objects. Through the DOM, developers can manipulate the content, structure, and styles of a webpage dynamically using programming languages like JavaScript.

How do you select an element from DOM ?

Use querySelector() and querySelectorAll() for modern, flexible selection with CSS selectors. Use getElementById() for unique element selections by id. Use getElementsByClassName() and getElementsByTagName() when you need collections of elements by class or tag, but be aware these are live collections, meaning they update dynamically when the DOM changes.
